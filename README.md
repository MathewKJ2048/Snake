# Snake
The snake game

Instructions:
Run main() in class Field to play the game.
Reset JVM every time before main() is run.
Graphics and controls can be customized by changing the appropriate state variables in class Field.

'w'-up
's'-down
'a'-left
'd'-right
'e'-pause/play
'q'-quit
